Diseased Plant Detection Using Image Processing

This team project aims to help farmers detect plant diseases early using image processing and AI. The system captures real-time images of plant leaves, processes them, and uses CNNs and YOLOv10 to identify diseases. When a disease is detected with high confidence, an Arduino Uno triggers an LED to alert the user, making the system interactive and suitable for field use.Our solution supports early action, helping reduce crop loss and improve farm productivity through smart technology.
